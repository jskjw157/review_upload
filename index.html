Cafe24 Local Review Manager

Electron + Node.js + TypeScript 기반의 로컬 리뷰 자동 등록 앱 프로토타입

로그아웃 상태

단건 리뷰 등록

상품 선택 후 텍스트, 별점, 이미지(옵션)를 입력해 리뷰를 등록하세요.

인증 & 토큰 관리

  • OAuth 2.0으로 로그인하고 토큰을 로컬에 저장합니다.
  • 토큰 만료 시 자동 갱신 로직을 처리합니다.
  • API 호출 시 액세스 토큰을 포함해 리뷰를 등록합니다.

자동 업로드 모드

CSV / XLSX 파일을 선택해 여러 리뷰를 한 번에 업로드합니다.

업로드 로그

  • 아직 업로드 기록이 없습니다.

네트워크 오류 재시도, 토큰 만료 시 갱신, 다크/라이트 모드와 같은 추가 기능을 염두에 둔 프로토타입입니다.
